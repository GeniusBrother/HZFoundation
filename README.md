Installation
==============
### CocoaPods

1. Add `pod 'HZURLManager` to your Podfile.
2. Run `pod install` or `pod update`.
3. Import \<HZURLManager/HZURLManager.h\>.

Documentation
==============
Full API documentation is available on [CocoaDocs](http://cocoadocs.org/docsets/HZFoundation/).<br/>

==============
